# Инструкция по настройке бота для приёма Excel файлов

## Проблема
Telegram Mini Apps на телефоне не могут напрямую скачивать файлы. Нужен бот, который примет данные и отправит файл в чат.

## Решение

Есть 2 варианта:

### Вариант 1: Простой (без backend) - РЕКОМЕНДУЮ СЕЙЧАС

**Используем кнопку "Поделиться" вместо скачивания:**

На телефоне:
1. Нажмите "Выгрузить в Excel"
2. Данные скопируются в буфер обмена
3. Вставьте их в любой чат или заметку
4. Сохраните как текстовый файл
5. Откройте в Excel

Преимущества:
- ✅ Работает сразу, без настройки бота
- ✅ Не нужен сервер
- ✅ Простота

Недостатки:
- ❌ Нужно вручную вставить текст

---

### Вариант 2: С backend (полноценное решение)

Нужно развернуть бот-сервер, который:
1. Принимает данные от Mini App через `tg.sendData()`
2. Создаёт CSV файл
3. Отправляет файл пользователю в Telegram

**Требуется:**
- Node.js сервер (или Python)
- Хостинг (Heroku, Railway, VPS)
- Настройка webhook бота

**Инструкция:**

1. Создать бот-сервер на Node.js:
```javascript
const TelegramBot = require('node-telegram-bot-api');
const token = '8680275148:AAEctDBvQRax5FXQKh5pf00p-8GSdOVlk7Q';
const bot = new TelegramBot(token, {polling: true});

bot.on('message', (msg) => {
    if (msg.web_app_data) {
        const data = JSON.parse(msg.web_app_data.data);
        
        if (data.action === 'export_excel') {
            // Декодируем base64
            const csvContent = Buffer.from(data.data, 'base64').toString('utf-8');
            
            // Создаём файл
            const buffer = Buffer.from(csvContent, 'utf-8');
            
            // Отправляем пользователю
            bot.sendDocument(msg.chat.id, buffer, {}, {
                filename: data.filename,
                contentType: 'text/csv'
            });
        }
    }
});
```

2. Развернуть на хостинге
3. Бот автоматически будет отправлять файлы
