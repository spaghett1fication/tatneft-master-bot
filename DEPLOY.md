# Инструкция по развертыванию Telegram Mini App

## Что нужно для запуска

1. **Telegram Bot** - бот, к которому будет привязано приложение
2. **Хостинг** - место где будут размещены HTML/CSS/JS файлы
3. **HTTPS** - обязательно для Mini Apps (Telegram требует защищённое соединение)

---

## Шаг 1: Создание бота в Telegram

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Введите имя бота (например: "Отчеты Татнефть")
4. Введите username бота (например: `tatneft_reports_bot`)
5. Сохраните **токен бота** - он понадобится позже

---

## Шаг 2: Загрузка файлов на хостинг

### Вариант А: GitHub Pages (бесплатно)

1. Файлы уже загружены на GitHub: https://github.com/spaghett1fication/tatneft-master-bot

2. Включите GitHub Pages:
   - Зайдите в Settings репозитория
   - Раздел "Pages" (слева в меню)
   - Source: выберите `main` branch и папку `/ (root)`
   - Нажмите Save
   - Через пару минут приложение будет доступно по адресу:
     ```
     https://spaghett1fication.github.io/tatneft-master-bot/
     ```

### Вариант Б: Vercel (бесплатно, проще)

1. Зайдите на [vercel.com](https://vercel.com)
2. Войдите через GitHub аккаунт
3. Нажмите "New Project"
4. Выберите репозиторий `tatneft-master-bot`
5. Нажмите "Deploy"
6. Через минуту получите URL вида: `https://tatneft-master-bot.vercel.app`

### Вариант В: Netlify (бесплатно)

1. Зайдите на [netlify.com](https://netlify.com)
2. Drag & Drop папку `C:\bat` в окно браузера
3. Получите URL вида: `https://random-name.netlify.app`

---

## Шаг 3: Привязка Mini App к боту

1. Вернитесь к [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newapp`
3. Выберите вашего бота из списка
4. Введите название приложения: `Отчеты мастеров`
5. Введите описание: `Приложение для отправки отчетов о работе`
6. Загрузите иконку (любую PNG 640x640)
7. **Введите URL вашего приложения** (из шага 2):
   ```
   https://spaghett1fication.github.io/tatneft-master-bot/
   ```
   или
   ```
   https://tatneft-master-bot.vercel.app
   ```
8. Введите Short name (латиница): `reports`
9. Отправьте скриншот приложения (можно взять из папки `картинка 1.png`)

---

## Шаг 4: Открытие Mini App

После создания:

1. BotFather даст вам **прямую ссылку** на Mini App вида:
   ```
   https://t.me/tatneft_reports_bot/reports
   ```

2. Можно также добавить кнопку-меню в бота:
   - Отправьте BotFather команду `/mybots`
   - Выберите бота → Bot Settings → Menu Button
   - Configure Menu Button → URL: вставьте прямую ссылку на Mini App
   - Text: `Открыть отчеты`

3. Теперь пользователи смогут открыть приложение:
   - По прямой ссылке
   - Через кнопку меню бота
   - Через команду `/start` в боте

## Как работает приложение

### Сохранение данных
- Все отчеты сохраняются **локально** в браузере пользователя (localStorage)
- Данные не удаляются при закрытии приложения
- Работает офлайн

### Экспорт в Excel
- Кнопка "Выгрузить в Excel" создаёт CSV файл
- Файл всегда называется `tatneft_reports.csv` (перезаписывается при повторной выгрузке)
- Открывается в Excel с корректной кириллицей
- Структура совпадает с вашим шаблоном (колонки: Дата, Объект, Мастер, Событие, АвтоМ и т.д.)

## Настройка объектов и видов работ

Чтобы изменить список объектов или видов работ:

1. Откройте файл `index.html`
2. Найдите секцию `<select id="object">` для объектов
3. Найдите секцию `<select id="workType">` для видов работ
4. Измените `<option>` по образцу:
   ```html
   <option value="obj4">Объект 4</option>
   ```
5. Сохраните и загрузите на хостинг

---

## Дополнительные возможности

### Добавление логотипа Татнефть
В `index.html` можно добавить после `<h1>`:
```html
<img src="logo.png" alt="Татнефть" style="width: 120px; margin: 0 auto; display: block;">
```

### Отправка данных на сервер
Сейчас данные хранятся локально. Для отправки на сервер нужно:
1. Создать backend API (Node.js, Python, PHP)
2. Добавить в `app.js` функцию отправки через `fetch()`
3. Хранить данные в базе данных

## Проверка работы

1. Откройте бота в Telegram на телефоне
2. Нажмите кнопку меню или перейдите по ссылке
3. Заполните форму
4. Нажмите "Отправить отчет"
5. Проверьте, что отчет появился в списке
6. Нажмите "Выгрузить в Excel" - должен скачаться CSV файл

## Поддержка и обновления

Для обновления приложения:
1. Измените файлы локально
2. Закоммитьте изменения в Git:
   ```bash
   git add .
   git commit -m "Обновление функционала"
   git push
   ```
3. GitHub Pages/Vercel автоматически обновят приложение
