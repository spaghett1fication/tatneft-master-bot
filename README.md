# Telegram Mini App - Отчеты мастеров Татнефть

Прототип для практики в Татнефть Инжиниринг

## Особенности

✅ **Простой интерфейс** - подходит для возрастных пользователей
✅ **Офлайн-режим** - работает без интернета, данные сохраняются локально
✅ **Легковесный** - минимальный трафик, работает на слабых Android устройствах
✅ **Валидация** - проверка формата гос. номеров и часов работы
✅ **LocalStorage** - все данные хранятся в телефоне до отправки

## Как запустить

### Вариант 1: Локальный сервер (для теста)

```bash
# Установи любой простой HTTP сервер
npm install -g http-server

# Запусти в папке C:\bat
http-server -p 8080
```

Открой в браузере: http://localhost:8080

### Вариант 2: Telegram Mini App (реальный тест)

1. Открой [@BotFather](https://t.me/botfather) в Telegram
2. Создай нового бота: `/newbot`
3. Настрой Mini App: `/newapp`
4. Залей файлы (index.html, styles.css, app.js) на хостинг
5. Укажи URL в BotFather

**Бесплатные хостинги для теста:**
- GitHub Pages (github.com)
- Vercel (vercel.com)
- Netlify (netlify.com)

## Что сделано

- ✅ Форма с 6 полями (дата, ФИО, гос.номер, часы, объект, вид работ)
- ✅ Валидация гос. номеров (А123БВ 116)
- ✅ Валидация часов (0-24)
- ✅ Локальное сохранение всех отчетов
- ✅ Просмотр сохраненных отчетов
- ✅ Работа без интернета
- ✅ Уведомления о статусе связи
- ✅ Адаптация под Telegram тему (светлая/темная)

## Технические детали

- **Размер**: ~6 КБ (минимальный трафик)
- **Зависимости**: только Telegram Web App API
- **Хранение**: LocalStorage (работает офлайн)
- **Совместимость**: все Android устройства с браузером

## Как тестировать

1. Открой в браузере телефона
2. Заполни форму
3. Отключи интернет
4. Попробуй сохранить еще один отчет
5. Включи интернет обратно
6. Проверь что все отчеты на месте
